习惯的记分卡

日本铁路系统被誉为世界上最好的系统之一。如果你在东京坐火车，你会注意到列车员有一个奇特的习惯。

当每个操作员驾驶火车时，他们会进行一种指向不同物体的仪式并喊出指令。当火车接近信号时，操作员会指向它并说："信号是绿色的。"当火车拉入车站时，操作员会指向速度表并喊出确切的速度。当需要离开时，操作员会指向时间表并说明时间。

在月台上，其他员工正在执行类似的动作。离开前，工作人员会指向平台边缘并宣布："一切就绪！"每个细节都被识别、指向并大声说出。

这个过程，被称为指向和呼叫（Pointing-and-Calling），是一个旨在减少错误的安全系统。它看起来很傻，但它的效果令人难以置信。
